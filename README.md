# homebrew-gor-mobile

[![formula](https://img.shields.io/github/v/release/gorban-dev/gor-mobile?label=formula&color=orange)](https://github.com/gorban-dev/gor-mobile/releases)
[![license](https://img.shields.io/github/license/gorban-dev/homebrew-gor-mobile)](./LICENSE)
[![install](https://img.shields.io/badge/brew-install%20gorban--dev%2Fgor--mobile%2Fgor--mobile-orange)](https://github.com/gorban-dev/gor-mobile)

Homebrew tap for [gor-mobile](https://github.com/gorban-dev/gor-mobile).

## Install

```sh
brew install gorban-dev/gor-mobile/gor-mobile
gor-mobile init
```

## Upgrade

```sh
brew update && brew upgrade gor-mobile && gor-mobile update
```

## Uninstall

```sh
gor-mobile uninstall
brew uninstall gor-mobile
brew untap gorban-dev/gor-mobile
```

The `Formula/gor-mobile.rb` file is regenerated by the `release.yml` workflow in
`gorban-dev/gor-mobile` on every version tag.
